Dealer

A simple app designed for HowdyHack 2020

DevPost

About

The web-app for Dealer is written in Angular, and has an express/node js backend with a MongoDB atlas cluster as it's storage.
Both parts of the web app come pre-built and ready to deploy with one command and can be ran on virtually any machine with a screen!

The other part of the app is the IOS application that was written in Swift, and pulls info from the backend regarding specific users, granted that they are on the same network

Be sure to check out our DevPost to see more about the app and it's code.
